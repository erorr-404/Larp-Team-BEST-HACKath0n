# Папка DataConverter
Тут мають бути класи, які будуь перетворювати первинно ороблені дані у зручний для використання у 3Д візуалізації формат.

# Зучний формат:
Це має бути масив із структур (або не зовсім). Він має містити дані:
    
    - Часова позначка

    - Координати X, Y, Z у Декартовій системі відносно стартової точки (ENU)
    - Кут нахилу у цій точці (Кватеріон)

    - Лінійна швидкість апарату в цей момент
    - Лінійне прискорення апарату в цей момент

    - Кутова швидкість апарату в цей момент
    - Кутове прискорення апарату в цей момент
